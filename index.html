Positionneur d'évènement

Ce programme écrit en JavaScript avec les bibliothèques D3.js, jQuery et jQueryUi vous permet de cliquer sur une carte du monde pour générer une légende automatique. Elle séparera les pays cliqués en 4 groupes égaux, calculés avec les quartiles. Vous pouvez ainsi visualiser géographiquement ce que vous souhaitez. Un tableau sur la droite vous indiquera les valeurs numériques correspondant à vos clics, triés par ordre décroissant.

Note: Ctrl + clic pour faire baisser la valeur d'un pays.
